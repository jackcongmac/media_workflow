# 出片操作手册：剪映一站式（即梦/Seedance）｜手把手

适用：本账号每一集。以 NX-071 为例。你有**国内版剪映 VIP**，里面自带 **即梦/Seedance** 出图+出视频。
**主路：全部在剪映里做**（出图→图生视频→拼接→配音→字幕→质感→导出）。
**备用**：某个镜头剪映出得不满意，再用**可灵铂金会员**单独补那一个。

素材来源：
- 出图用 `NX-071_undelivered_letter/image_prompts.md`（FF01–FF12）
- 出视频用 `NX-071_undelivered_letter/video_prompts.md`（V01–V12）
- 拼接质感用 `NX-071_undelivered_letter/texture_notes.md`

---

## 第一步：在剪映里出 12 张首帧图（即梦 AI 作图）
> 入口名称可能是「AI 作图 / 即梦 / 图片生成」。找不到就把界面截图发我，我按实际按钮校准。

对 FF01…FF12 重复：
1. 打开剪映 → 找到 **AI 作图 / 即梦** 入口 → 文生图。
2. 提示词粘贴对应 FFxx，末尾补全局风格：
   `竖屏9:16，夜间监控画风，低照度，写实，电影感冷调，画面内无任何文字/水印/logo，监控/夜视/噪点质感`
3. 比例 **9:16**，生成多张里挑最"暗、脏、监控感"的一张。
4. 保存到 `03_images/`，命名 **FF01.png … FF12.png**。

## 第二步：在剪映里图生视频（Seedance）
> 入口可能是「AI 视频 / 图生视频 / Seedance」。

对 V01…V12 重复：
1. 选 **图生视频**，上传对应首帧（V01 配 FF01，依次类推）。
2. 提示词粘贴对应 Vxx（运动描述）。
3. 时长按下表；模型选最新 Seedance / 高品质。
4. 满意就用，乱动/穿模就改"更慢、更小幅、镜头几乎不动"重生成。
5. 导出/保存的片段放 `05_video_projects/`，命名 **V01.mp4 … V12.mp4**。
   （如果直接在剪映时间线上生成，也可不单独导出，直接进第三步拼接。）

## 第三步：拼接 + 质感（详见 texture_notes.md）
1. 9:16 / 1080×1920 工程，把 12 段按顺序排上主轨，剪到目标时长（总 ~72s）。
2. 加：监控宫格 / 时间码(含 03:17 卡顿) / 夜视绿 / 噪点 / 扫描线 / 暗角。
3. 文本朗读：贴 `voiceover.txt`，冷静低沉男声。
4. 导入 `captions.srt` 字幕，对齐。
5. 对讲机三声音效 + 低音量雨声/BGM（仅可商用授权）。

## 第四步：导出 + 校验
1. 导出 1080×1920，码率较高，音量约 -14 LUFS，**勾选「AI 生成内容」标识**。
2. 命名 `final.mp4`，放进 `NX-071_undelivered_letter/`。
3. 终端校验：
   `cd /Volumes/WorkHD/media_testing && ./tools/validate_export.sh shorts/yejiban_anomaly_archive/NX-071_undelivered_letter/final.mp4`
   看到 `PASS` 即合格。
