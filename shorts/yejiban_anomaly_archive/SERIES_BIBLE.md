# 《今晚值夜班》世界观圣经 (SERIES BIBLE)

> 旗舰格式 = 中式异常档案宇宙的"夜班监控审核员"视角。本文件是所有 NX 档案集的
> 唯一事实来源；写任何一集前先读这里，写完对照这里自查。
> 设计依据：`docs/superpowers/specs/2026-06-13-anomaly-archive-universe-design.md`

---

## 1. 账号设定（谁在讲）

- **宇宙**：**不存在事务所**——一个**非官方**民间机构，专门处理那些没法报警、
  上不了新闻、但确实发生过的事。强调"非官方"是为了规避假新闻/拟真冒充风险。
- **旗舰叙述者**：事务所的**夜班监控审核员**。每晚审看全国各地上传的异常监控片段，
  按规定**只能记录、不能干预**。
- **人设关键词**：疲惫、克制、打工人、见过太多、嘴上吐槽心里发凉。**不露脸**
  （只露手、键盘、监控墙、保温杯）。

---

## 2. 招牌开场句（连载按钮 · 一字不差）

```
这是我值夜班第 ___ 天，今晚这段监控，我看了三遍。
```

- 永远是**第一句**（0–4s 内出现）。
- `___` 填当集天数；天数**跨集累计**，制造连载追更感。
- 起源集用"第1天"。

## 3. 规则梗（评论区按钮 · 模板）

```
如果你……请不要……
```

- 每集由审核员**念出该异常的"规则"**，是给观众的禁忌警告。
- 必须**可被复制仿写**——评论区抄不动，就出不了圈。
- 句式短、像咒语、留想象空间。例：`如果你深夜收到一封没有寄件人的信，请不要拆开，先看邮戳，是哪一年。`

## 4. 规格

| 项 | 标准 |
|---|---|
| 时长 | 60–75 秒 |
| 画幅 | 竖屏 1080×1920 |
| 调性 | **怪谈外壳 + 情感/道德反转内核**（不是纯吓人；"看完想转给某个人"才出圈） |
| 配音 | AI TTS，冷静、克制的夜班旁白 |
| 字幕 | 烧录中文，每行 7–12 字，每镜只高亮一句关键 |
| 响度 | 目标 ≈ -14 LUFS |

## 5. 质感系统（天然吸收 AI 瑕疵）

监控宫格 / 时间码 / 夜视绿光 / 失真音频 / 保安室屏摄 / 对讲机噪声 / 低帧率 / 噪点 /
扫描线 / 遮挡。人物可远、可小、动作不必完美——**刻意的低质感正是掩盖 AI 破绽的手段**。
**优先用剪映/即梦自带 AI 出视频**，单镜头质感撑不住电影感时再补可灵。

## 6. Beat 结构（60–75s）

| 时间 | 内容 |
|---|---|
| 0–4s | 招牌开场句 + 最怪一帧 |
| 4–15s | 审核员吐槽式介绍（"一开始我以为是系统 bug"） |
| 15–35s | 异常重复出现 → 形成"规则"（念出"如果你…请不要…"） |
| 35–60s | 发现它不是随机，而是在等某个人 / 完成某件未了的事 |
| 60–70s | 情感反转（怪物 = 人间的亏欠 / 执念 / 守护） |
| 70–75s | 归档，抛出下一条异常编号（评论区接梗） |

## 7. 世界规则 / 复现符号（养 B站 考据 + 二创）

1. **NX 编号**：每集一个档案号 `NX-071`、`NX-072`…（连续累计）。
2. **03:17**：监控时间永远在 03:17 卡一下（异常的时间锚点）。
3. **对讲机三声**：异常出现前，对讲机会响三声。
4. **7号屏**：监控墙上 7 号屏永远最危险；**审核员不能离座**（离座会被下一个监控拍到）。
5. **红色三角贴纸**：贴在档案上 = "已回访"。
6. **寻人启事伏笔**：画面角落反复出现**同一张寻人启事**（跨集长线，留给考据党）。

> 每集至少自然植入 3 个符号；寻人启事建议每集都在角落出现一次。

## 8. 调性红线（合规 · 不可越）

- **主动打 AI 标**（国标 2025-09-01 起强制；打标不降权，不打标才被限流）。
- **不做假新闻、不拟真冒充**真实机构/真人/真实事件；用"不存在事务所"虚构框架兜底。
- **不用真实地名/真人姓名**；县城、地铁、小区一律匿名化（"某县""某线""某小区"）。
- 史实/现实细节若不确定，不得当确定事实陈述。
- 不碰低俗猎奇。
