## 1. 战略定位

| 维度 | 决定 |
|---|---|
| 平台 | 抖音 / 小红书 / B站（国内、中文受众） |
| 目标 | **爆流量 / 出圈**（不急变现，偏娱乐/情绪/强反转） |
| 硬约束 | **纯电影级 AI 视频**：真实运动镜头，拒绝静图拼接、拒绝国画风 |
| 呈现 | 无人脸友好；主动打 AI 标（合规）；不做假新闻/拟真冒充框架 |
| 核心打法 | 靠**可复制的格式**，不赌单条；把"AI 的不真实"变成怪谈质感的**卖点** |

**关键洞察（双信源一致：网络研究 + Codex 独立判断）**：
在国内追求"AI 拍得像真电影"是死路（真人脸/手/口型做不好，且被骂"AI 味"）。
正确做法是做**"没有 AI 就不可能存在"的内容**，让超现实/诡异成为卖点。

**三条出圈铁律**：
1. 必须有**一句能被复述/玩梗的话**（评论区抄不动就火不了）。
2. **无人脸、短镜头、用能吸收瑕疵的质感**（监控/新闻/档案）盖住 AI 破绽。
3. **主动打 AI 标，但别把"AI"当卖点**。国标自 2025-09-01 起强制标识；被限流的是
   *不打标* 和 *低质猎奇*，不是打标的高质内容（《纸手机》打标仍 100M+ 被央媒转）。

**预期校准**：研究显示纯 AI 出圈是"尖峰式"的（真人+AI 才是稳定主流），但《纸手机》
证明纯 AI 能爆。因此用"格式引擎"摊薄单条风险，而非赌爆款。

## 2. 账号世界观（统一宇宙 = 中式 SCP）

虚构的 **"不存在事务所"**：专门处理那些没法报警、上不了新闻、但确实发生过的事。
**非官方机构**（规避假新闻风险），由民间档案员维护。所有格式共用同一宇宙、
编号系统与复现符号，从而长出 B站 考据 + 二创生态。

---

## 3. A+B 双引擎

- **A（日更引擎）= 《今晚值夜班》**：稳定产出、连载、出圈。本 spec 的主体。
- **B（偶发大招）= 情感微电影**（《纸手机》模式）：蹭节点（清明/年关）用同一宇宙的
  某个"档案"展开成 3–5 分钟短片，冲爆款 + 争取央媒级转发。后续单独立 spec。
- **衍生格式（后期换口味、二次出圈）**：
  - 《不存在事务所》（民间档案员视角，禁忌规则向，开场"如果你……请不要……"）
  - 《地方台重演》（虚构地方台重演栏目，黑色幽默 + 道德反转，最安全合规）

## 4. 旗舰格式规范 ——《今晚值夜班》

**设定**：叙述者是"不存在事务所"的**夜班监控审核员**，每晚看全国各地异常监控，
按规定只能记录、不能干预。一人 + 一间监控室 + 全国素材 = 最强日更引擎、最盖 AI、
最具打工人共鸣。

### 4.1 招牌句与梗
- **招牌开场句（连载按钮）**：
  `这是我值夜班第 ___ 天，今晚这段监控，我看了三遍。`
- **嵌入的可复制梗（评论区按钮）**：每集审核员念出该异常的"规则"——
  `如果你……请不要……`

### 4.2 规格
- **时长**：60–75 秒
- **调性**：怪谈外壳 + **情感/道德反转内核**（非纯吓人——"看完想转给某个人"才出圈）
- **质感系统**：监控宫格 / 时间码 / 夜视绿光 / 失真音频 / 保安室屏摄 / 对讲机噪声。
  人物可远、可小、动作不必完美——刻意的低帧率、噪点、遮挡、压缩**吸收 AI 瑕疵**。
- **配音**：AI TTS，冷静、克制的夜班旁白。
- **字幕**：烧录中文字幕，每行 7–12 字，每镜只高亮一句关键。

### 4.3 Beat 结构（60–75s）
| 时间 | 内容 |
|---|---|
| 0–4s | 模板开场句 + 最怪一帧 |
| 4–15s | 审核员吐槽式介绍（"一开始我以为是系统 bug"） |
| 15–35s | 异常重复出现 → 形成"规则"（念出"如果你…请不要…"） |
| 35–60s | 发现它不是随机，而是在等某个人 / 完成某件未了的事 |
| 60–70s | 情感反转（怪物 = 人间的亏欠 / 执念 / 守护） |
| 70–75s | 归档，抛出下一条异常编号（评论区接梗） |

### 4.4 世界规则 / 复现符号（养考据 + 二创）
- 档案编号：`NX-071`、`NX-214` …（连续累计）
- 监控时间永远在 `03:17` 卡一下
- 异常出现前，对讲机响三声
- `7号屏` 永远最危险；审核员不能离座（离座会被下一个监控拍到）
- 红色三角贴纸 = "已回访"
- 画面角落反复出现同一张寻人启事（跨集伏笔）

## 5. 制作工作流（纯 AI 电影级）

```
选题/源料研究 + 历史与逻辑 QA
  → 脚本（写作室：5钩子/反转/规则句）
  → 分镜 6–8 镜
  → 首帧参考图
  → 图生视频短镜头（3–6s / 镜，不做一条长视频）
  → AI 配音（冷静夜班旁白）
  → 烧录字幕
  → 质感层（监控/噪点/时间码/夜视/对讲机噪声）
  → FFmpeg 校验（1080×1920、响度、时长、码率）
  → 质量门
  → 打 AI 标
  → 发布包
```

**工具链（已与用户确认：剪映 VIP 为主，省钱优先）**：
用户持有**国内版剪映 VIP**，覆盖本流水线大部分环节，首轮力求接近零额外花费。

| 环节 | 工具 | 备注 |
|---|---|---|
| 剪辑/拼接/调色 | 剪映 VIP | 主力 |
| 自动字幕 | 剪映 VIP | 识别 + 烧录 |
| AI 配音(TTS) | 剪映 VIP | 选冷静夜班旁白音色 |
| 质感层（噪点/老电视/监控/夜视/时间码/转场） | 剪映 VIP | 我们的赛道刻意低质感，剪映特效大多够用 |
| 封面 | 剪映 VIP / 即梦 | |
| 首帧参考图 | 即梦（字节系，与剪映同生态）/ GPT Image | 按需 |
| **核心：电影级图生视频短镜头** | **先用剪映/即梦自带 AI 试**；质感撑不住再**单点补**即梦/可灵 Keling | 怪谈/监控质感天然盖 AI 瑕疵，剪映自带很可能够用 |
| 校验/批处理 | FFmpeg（1080×1920、响度、时长、码率） | Codex 自动化 |

**关键判断**：本赛道是怪谈/监控质感（刻意低帧率、噪点、夜视、遮挡），本就在掩盖 AI 破绽，
因此剪映自带 AI 视频很可能"够用"，不必先买可灵。**只在单镜头质感撑不住电影感时按需补**。

- **首轮预算**：力求接近 ¥0 额外花费（剪映 VIP 已付）；仅按需单点补图生视频，远低于旧文档 $100 上限。

**质量门（发布前全为真才发）**：竖屏 1080×1920；字幕不裁切；无穿帮现代物/错字；
AI 动作无明显穿模/漂移（监控质感下容忍度更高）；史实不当确定陈述需标注；
前 3 秒有钩子；结尾 5 秒引评论；已打 AI 标。

## 8. 首条样片 ——《今晚值夜班 · 第1天》（起源集，NX-071）

**起源设定**：老员工留下三条规矩就走了——
`7号屏别盯太久`、`03:17 会卡一下别管`、`对讲机响三声就归档`。

**事件**：凌晨 03:17，7号屏卡了一下——一个**邮递员**，正在送一封 **20 年前（2006）**
没送到的信。

**反转**：他 20 年前送信途中出事，没能把那封信送达；此后每晚补送那一封，
寄给当年那个一直在等信、如今已长大的孩子。它不害人——是执念与亏欠。

**规则梗（评论区按钮）**：
`如果你深夜收到一封没有寄件人的信，请不要拆开，先看邮戳，是哪一年。`

**编号**：NX-071
**结尾钩**：下一条 NX-072——7号屏右下角那张寻人启事，是谁？

## 9. 验证计划（首轮）

先做 **3–5 条**《今晚值夜班》，同格式不同档案，作为格式验证：
- 全部打 AI 标、私密/小范围预览先测。
- 测量：前 3 秒留存、完播率、评论率、是否有人在评论区仿写"如果你……请不要……"、
  是否有人问"NX 编号 / 03:17 / 寻人启事"（考据信号）。
- 决策：留存强 → 进入连载；评论玩梗强 → 加重规则句与世界符号；
  情感反转转发强 → 排期一条 B 引擎情感微电影冲爆款。
